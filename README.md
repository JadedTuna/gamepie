# GamePie
GamePie is a game engine written in Python.

# Purpose
It was written to ease creation of cross-platform games
using Python. Currently it supports:

* Linux
* iOS (Pythonista)
* Window (Not tested)
* Mac (Not tested)
* Android (QPython && PyGame)
